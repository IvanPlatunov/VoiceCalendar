# 🗓️ VoiceCalendar

Голосовой помощник для управления задачами. Работает в фоновом режиме, активируется по кодовому слову, понимает русскую речь.

---

## 🚀 Быстрый старт

1. Установите Python 3.10+ с [python.org](https://python.org)
2. Скачайте и распакуйте архив с программой
3. Дважды кликните `start.bat` (Windows) или `start.sh` (Mac/Linux)
4. Скажите **«Привет, календарь»** и команду

При первом запуске автоматически установятся все зависимости.

Поддерживаются:

- **Относительные:** сегодня, завтра, послезавтра

## 💾 Где хранятся задачи

По умолчанию задачи сохраняются в файл `calendar_data.json` рядом с программой.

### Подключить Google Calendar

1. Создайте проект в [Google Cloud Console](https://console.cloud.google.com)
2. Включите Google Calendar API
3. Создайте OAuth 2.0 Client ID (тип: Desktop app)
4. Скачайте `credentials.json` в папку с программой
5. Откройте `.env` в блокноте и измените:

```bash
VC_STORAGE_TYPE=google
VC_GOOGLE_CREDENTIALS=credentials.json
```

6. При первом запуске откроется браузер для авторизации

## 🔧 Настройка

Все настройки в файле `.env`:

```bash
VC_STORAGE_TYPE=json          # json или google
VC_LANGUAGE=ru-RU             # язык распознавания
VC_SPEECH_RATE=150            # скорость речи (50-300)
VC_SPEECH_VOLUME=1.0          # громкость (0.0-1.0)
VC_SAMPLERATE=16000           # качество записи
VC_SILENCE_THRESHOLD=300      # чувствительность микрофона
VC_SILENCE_LIMIT=1.5          # пауза перед окончанием фразы
```

## ❓ Частые проблемы

**Не слышит микрофон**
- Проверьте, что микрофон подключён и работает
- Уменьшите `VC_SILENCE_THRESHOLD` до 100

**Не распознаёт речь**
- Говорите чётче и громче
- Проверьте интернет-соединение

**Не говорит голосом**
- Установите русские голоса в настройках Windows

## 📦 Зависимости

Устанавливаются автоматически. Для справки:

- `sounddevice` — запись с микрофона
- `numpy` — обработка звука
- `requests` — отправка речи на распознавание
- `pyttsx3` — синтез голоса
- `google-api-python-client` — Google Calendar (опционально)
