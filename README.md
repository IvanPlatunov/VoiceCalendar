# 🗓️ VoiceCalendar

Голосовой помощник для управления задачами через голосовые команды. Работает в фоновом режиме, активируется по кодовому слову, поддерживает русский язык.

## 🚀 Быстрый старт

### Требования
- Python 3.10 или выше ([скачать](https://python.org))
- Рабочий микрофон
- Интернет-соединение для распознавания речи

### Установка и запуск

#### Windows

1. Скачайте и распакуйте архив с программой
2. Запустите файл `start.bat` двойным щелчком
3. Дождитесь автоматической установки зависимостей
4. Скажите **«Привет, календарь»** и дайте команду

#### macOS/Linux

1. Скачайте и распакуйте архив с программой
2. Откройте терминал в папке с программой
3. Выполните команду:

   ```bash
   chmod +x start.sh && ./start.sh
   ```

4. Дождитесь автоматической установки зависимостей
5. Скажите **«Привет, календарь»** и дайте команду

> **Примечание:** При первом запуске все необходимые библиотеки устанавливаются автоматически.

## 🎤 Голосовые команды

### Активация помощника

Произнесите одно из кодовых слов:
- «Привет, календарь»
- «Календарь»
- «Ассистент»

### Управление задачами

| Действие | Примеры команд |
|----------|----------------|
| **Добавить задачу** | «Поставь задачу встреча на завтра в 15:00» |
| **Добавить с приоритетом** | «Запиши срочно позвонить маме сегодня в 18:00» |
| **Показать задачи** | «Покажи задачи на сегодня», «Что у меня на завтра» |
| **Удалить все задачи** | «Очисти всё» |
| **Получить справку** | «Помощь», «Что ты умеешь» |
| **Фоновый режим** | «Стоп», «Фон», «Пауза» |
| **Выход из программы** | «Выход», «Пока» |

## 📅 Распознавание дат и времени

Программа понимает различные форматы дат и времени:

### Относительные даты
- сегодня
- завтра
- послезавтра

### Дни недели
- понедельник, вторник, среда, четверг, пятница, суббота, воскресенье

### Конкретные даты
- 15 января
- 3 марта
- 20 декабря

### Время
- в 15:00
- в 10:30
- в 9 часов
- в полдень (12:00)
- в полночь (00:00)

**Время по умолчанию:** 9:00 утра, если не указано иное.

## 🔴 Система приоритетов

Задачи могут иметь три уровня приоритета, которые отображаются цветом в Google Calendar:

| Приоритет | Ключевые слова | Цвет в Google Calendar |
|-----------|----------------|------------------------|
| **Высокий** | «срочно», «важно», «высокий» | 🔴 Красный |
| **Средний** | «средний» | 🟡 Жёлтый |
| **Обычный** | без указания приоритета | 🔵 Синий |

**Пример:** «Запиши срочно сдать отчёт в пятницу» — задача будет помечена как высокоприоритетная.

## 💾 Хранение данных

### Локальное хранилище (по умолчанию)

Задачи сохраняются в файл `calendar_data.json` в папке с программой.

### Интеграция с Google Calendar

Для синхронизации с Google Calendar выполните следующие шаги:

1. **Создайте проект в Google Cloud Console**
   - Перейдите на [Google Cloud Console](https://console.cloud.google.com)
   - Создайте новый проект

2. **Включите Google Calendar API**
   - В разделе «APIs & Services» → «Library»
   - Найдите и включите «Google Calendar API»

3. **Создайте учетные данные OAuth 2.0**
   - Перейдите в «APIs & Services» → «Credentials»
   - Нажмите «Create Credentials» → «OAuth client ID»
   - Выберите тип приложения: **Desktop app**
   - Скачайте файл `credentials.json`

4. **Настройте программу**
   - Поместите файл `credentials.json` в папку с программой
   - Откройте файл `.env` в текстовом редакторе
   - Измените следующие параметры:

     ```env
     VC_STORAGE_TYPE=google
     VC_GOOGLE_CREDENTIALS=credentials.json
     ```

5. **Авторизация**
   - При первом запуске откроется браузер
   - Войдите в свой Google аккаунт и предоставьте доступ к календарю
   - Токен авторизации сохранится автоматически

## 🔧 Настройка

Все настройки находятся в файле `.env`:

```env
# Тип хранилища: json (локально) или google (Google Calendar)
VC_STORAGE_TYPE=json

# Язык распознавания речи
VC_LANGUAGE=ru-RU

# Скорость синтеза речи (50-300 слов в минуту)
VC_SPEECH_RATE=150

# Громкость голоса (0.0 - 1.0)
VC_SPEECH_VOLUME=1.0

# Частота дискретизации записи (Гц)
VC_SAMPLERATE=16000

# Порог тишины (чувствительность микрофона)
VC_SILENCE_THRESHOLD=300

# Лимит тишины перед окончанием записи фразы (секунды)
VC_SILENCE_LIMIT=1.5
```

## ❓ Решение проблем

### Микрофон не реагирует

**Симптомы:** программа не слышит голосовые команды

**Решение:**
- Убедитесь, что микрофон подключен и работает
- Проверьте настройки микрофона в системе
- Уменьшите значение `VC_SILENCE_THRESHOLD` до 100-200 в файле `.env`

### Речь не распознается

**Симптомы:** программа слышит, но не понимает команды

**Решение:**
- Говорите четко и достаточно громко
- Проверьте интернет-соединение (требуется для распознавания речи)
- Убедитесь, что произносите команды после кодового слова

### Отсутствует голосовой вывод

**Симптомы:** программа работает, но не отвечает голосом

**Решение:**
- **Windows:** установите русские языковые пакеты в настройках системы
- **macOS/Linux:** убедитесь, что установлен `pyttsx3` и поддерживаемый синтезатор речи

### Ошибки при запуске

**Симптомы:** программа не запускается или выдает ошибки

**Решение:**
- Убедитесь, что установлена версия Python 3.10 или выше
- Проверьте, что все зависимости установлены: `pip install -r requirements.txt`
- Удалите папку `__pycache__` и файлы `.pyc`

## 📦 Зависимости

Все зависимости устанавливаются автоматически при первом запуске. Для справки:

### Основные библиотеки
- **sounddevice** — запись аудио с микрофона
- **numpy** — обработка аудиосигналов
- **requests** — отправка аудио на сервер распознавания речи
- **pyttsx3** — синтез речи (text-to-speech)

### Опциональные библиотеки
- **google-api-python-client** — интеграция с Google Calendar
- **google-auth-oauthlib** — авторизация OAuth 2.0 для Google API
